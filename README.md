# 16-bit MIPS Processor (VHDL)

## Description

This project implements a **basic 16-bit MIPS-like processor** written in **VHDL** and designed to run on an FPGA. The goal of the project is to demonstrate the core components of a processor datapath and control logic, including instruction fetching, decoding, execution, and memory access.

The design was developed as part of a **Computer Architecture laboratory project**.

## Features

* Custom **16-bit MIPS-style architecture**
* Modular VHDL design
* Separate units for datapath and control
* FPGA debugging using **seven-segment display**
* Button control using a **Mono Pulse Generator**

## Usage

The processor can be tested on an FPGA board by using switches and buttons to control execution. Internal signals and results can be visualized using the seven-segment display.

## Tools

* **VHDL**
* **Xilinx Vivado**
* **NexysA7 100T FPGA board**

## Purpose

This project demonstrates the implementation of a simple processor architecture in hardware and helps illustrate fundamental concepts such as instruction decoding, ALU operations, control logic, and datapath design.
